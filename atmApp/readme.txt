Проект - Банкомат (ATM).

Назначение: симуляция работы банкомата.

Структура:
atm.jar - исполняемой версия проекта;
atm.bat - .bat файл для запуска приложения;
accounts.txt - файл для хранения данных о картах клиентов банка;
atm.txt - файл для хранения значения наличия денежных средств в банкомате.

Структура accounts.txt:
Каждая строка содержит сведения о данных одной карты: "1111-2222-3333-4444 1111 1471901 false 0 1720108244199", где:
"1111-2222-3333-4444" - номер карты;
"1111" - ПИН-код;
"1471901" - баланс;
"false" - свдения о блокировке карты (true/false);
"0" - количество неверных вводов ПИН-кода;
"1720108244199" - время последней блокировки (в мс);

Особенности:
1. Карта блокируется после трех неверных попыток ввести ПИН-код.
2. м после верного ввода ПИН-кода.
3. Через сутки, в случае блокировки, количество попыток обновляется (карта разблокируется).

Для теста: номер карты 1111-2222-3333-4444, ПИН-код для нее 1111.
